# 서울시 공공자전거 따릉이 데이터 분석 & ML 프로젝트

서울시 따릉이 2025년 10~12월 대여 데이터(약 856만 건)를 활용한 머신러닝 프로젝트입니다.

## 프로젝트 요약

### 01. 이상치 탐지 (Anomaly Detection)

**목표**: 비정상 이용 패턴 및 수요 급변 시점 탐지

**방법**:
- **개별 건수**: Isolation Forest + LOF → 방치 의심 건 탐지
- **시계열**: STL 분해 후 잔차 기반 이상치 탐지

**결과**:

| 방법 | 탐지 결과 | 특징 |
|---|---|---|
| Isolation Forest | 4,000건 (2%) | 이용시간 평균 4배 길고 거리 짧음 (방치 의심) |
| LOF | 400건 (2%) | 밀도 기반 이상 패턴 |
| STL 시계열 | 19개 시간대 | 공휴일 이벤트 정확히 포착 |

**주요 탐지 이벤트**:

| 날짜/시간 | 탐지 내용 |
|---|---|
| 크리스마스 이브 18~19시 | 대여 **급증** (+46건) |
| 개천절·한글날·크리스마스 08시 | 출근 시간 대여 **급감** |
| 12월 6~7일 08시 | 한파로 인한 **급감** 추정 |
| 12월 31일 08~19시 | 연말 이동 패턴 이상 탐지 |

### 02. 시간별 대여 수요 예측 (Demand Forecasting)

**목표**: 각 대여소의 다음 1시간 대여 건수 예측 (자전거 재배치 최적화)

**모델**: LightGBM (회귀)

**피처**: 대여소, 시간/요일/월, 공휴일, 순환 인코딩(sin/cos), lag 피처(1h~720h), net_flow 래그

**결과**:

| 모델 | MAE | R² |
|---|---|---|
| 평균 베이스라인 | 4.31건 | -0.05 |
| **LightGBM** | **2.67건** | **0.541** |

- 베이스라인 대비 **MAE 38% 향상**
- 예측 10건 중 9건은 실제와 **±5대 이내**

**인사이트**: 날씨 피처 추가 실험 → 3개월 데이터에서는 학습/테스트 기간 날씨 분포 차이로 오히려 성능 저하 확인. 연간 데이터 확보 시 재적용 예정

---

### 03. 사용자 클러스터링 (User Clustering)

**목표**: 이용 패턴이 비슷한 사용자 그룹 분류 및 특성 파악

**모델**: K-Means (k=3 최적, Silhouette Score 기준)

**피처**: 이용시간, 이용거리, 속도, 시간대, 요일, 주말 여부, 나이

**결과**:

| 클러스터 | 비율 | 평균 이용시간 | 평균 거리 | 특징 |
|---|---|---|---|---|
| **장거리형** | 11% | 73분 | 6.5km | 평일, 속도 느림 (운동/관광 목적) |
| **레저형** | 21% | 16분 | 1.8km | 주말 집중 이용 |
| **단거리 평일형** | 67% | 12분 | 1.5km | 출퇴근, 가장 많은 비중 |

**인사이트**: 자전거 종류(bike_type) 피처 포함 시 전기자전거 여부로만 클러스터가 분리되는 현상 발견 → 제거 후 행태 기반 의미있는 그룹 분리 성공
